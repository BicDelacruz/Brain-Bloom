# Brain Bloom
C++ game made using the raylib library
https://www.raylib.com/

---------------------------------------------------------------------------------------------------------------------------------------------------------------------

1. Download as zip
2. Extract and open "main.code-workspace" file
3. Once in visual studio code, hit ctrl + f5 to run game
